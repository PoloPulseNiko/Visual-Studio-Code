Media Features

Not complete list

https://developer.mozilla.org/en-US/docs/Web/CSS/@media#media_types

Media types describe the general category of a device. Except when using the not or only logical operators, the media type is optional and the all type is implied.

hover

Does the primary input mechanism allow the user to hover over elements?

@media (hover: none) {
    body::before {
    display: block;
    content: 'This device doesn't support hover';
    }
}

orientation

Orientation of the viewport.

@media (orientation: landscape) {
    body::before {
    display: block;
    content: 'The device is in landscape orientation';
    }
}

width

Width of the viewport including width of scrollbar.

@media (width: 600px) {
    body::before {
    display: block;
    content: 'The viewport width is 600px';
    }
}
